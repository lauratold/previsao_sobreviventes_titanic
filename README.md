Este projeto de Ciência de Dados tem como objetivo prever a sobrevivência de passageiros do Titanic utilizando técnicas de Machine Learning. O modelo foi desenvolvido em Python com bibliotecas como Pandas, NumPy e Scikit-learn, aplicando etapas de pré-processamento, engenharia de atributos e treinamento supervisionado. Durante o projeto, foram criadas variáveis relevantes como tamanho da família, identificação de passageiros sozinhos, títulos sociais e deck da cabine, além do tratamento de dados ausentes e codificação de variáveis categóricas. O algoritmo escolhido foi o Random Forest Classifier, otimizado com GridSearchCV para melhorar a performance do modelo. Ao final, o sistema gera automaticamente um arquivo de submissão pronto para envio no Kaggle. Este projeto foi desenvolvido por Laura Toldo como prática de conceitos fundamentais de Ciência de Dados e Machine Learning.
